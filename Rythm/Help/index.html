Structure

Extract the contents to a folder on your computer. The archive contains folders: Help - help to work with the template; HTML version - all HTML files, Muse template - a template folder.

In the folder Muse template is the template Rhythm.muse, the folder of images - images, folder icons - ico and the assets folder - user style css.

How to add an image

1. Select the desired image.

2. In the Resources window (Ctrl+Shift+D) click on the image with the right mouse button and select Relink...

Editing text

1. Double-click on the text to edit it. Open the TEXT window (Ctrl+T). In the TEXT window you can change the font, size, color and other text settings.

To learn all the tricks of editing, go to the official website of Adobe Muse Help - https://helpx.adobe.com/muse/using/text-typography.html
